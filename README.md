# AI-Powered To-Do App using Antigravity

## Overview
This project demonstrates how to build a simple **To-Do web application using the Antigravity AI development tool**. Antigravity allows developers to use AI agents to automatically generate, test, and manage code. In this project, we use Antigravity to create a basic task management application.

The To-Do app allows users to:
- Add tasks
- Mark tasks as completed
- Delete tasks

This project shows how AI-assisted development can speed up application development.

---

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Simple user-friendly interface
- AI-assisted code generation using Antigravity

---

## Technologies Used
- Python
- Flask
- HTML
- CSS
- Antigravity AI Tool
---

## How Antigravity Was Used

The Antigravity AI tool was used to:

1. Generate the initial project structure
2. Create backend code using Flask
3. Generate HTML UI
4. Assist in debugging and testing the application
